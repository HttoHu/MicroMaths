## 1 开始
1. 程序的结构
	Merdog Hello world
	=======================
	program main
	{
		 print 'hello world';
	}.
	=======================

	program +程序名
	{
		...=>这里是你要添加的内容
	}. ->注意 有个点	
	程序名随便取
	如
	program Hey
	{
	}.
	这是个程序不执行任何操作
2. print 和字符串
	print 是打印的意思，可以输出字符串和变量的值
	如：
	print 'hey'; 点击运行后 会弹出一个窗口 会显示Hey.
	print '...'; ...为你需要显示的内容（注意不能出现中文和中文标点，否则程序会崩溃）
	别忘了后面要加分号；在merdog中，每一个语句都是要加分号的，这一点和许多语言是一致的。

## 2 使用变量
以下为Merdog支持的变量
	| Merdog| 对应的C++类型 
	| int	| long          整数
	| real	| double        小数
	| string| std::string   字符串
	| bool	| bool          布尔值
=> 变量定义
1. Type VarName := Expr;	Expr 为一个表达式,可以为一个值，也可以为一个算术表达式，定义变量一定要初始化
	如:
	real x:=2.3;
	x是变量名,real是x的类型表明x是一个浮点数（小数）类型 :=是赋值运算符，即将右边的值传递给左边
	翻译一下，创建一个小数变量，并且在其初始化为2.3
	int i:=3;
	定义一个整数变量，初始化为3
	int a:=2*3-3;
	TIPS=> 定义变量赋予变量的值一定要与其类型相吻合，原因见bug 1.
2. 多个变量定义共用一个类型中间用逗号隔开
	如:
	int a:=1,b:=2,c:=3;
=> 变量修改
    定义完变量后可使用:=对变量的值进行修改
	如:
	1. int a:=2;
	2. ...  一些代码
	3. a:=3;
	4. ...

	如果2中使用到了a，那个时候a的值为2;4中使用a，a的值为修改后的3，也就是修改仅对其后面的代码生效.
	更多例子:
	1. int a:=2;
	2. int b:=4;
	3. a:=b:=4;
	第三步会发生什么，上面我们介绍过了:=会将右边的值传给左边. 这样我们从右向左看，首先4传递给了b，b现在是4了，然后b又将现在的值传给了a,这样a也就是4了
